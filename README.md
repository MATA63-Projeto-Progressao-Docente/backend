# Backend

**Tecnologias**:
- Node.js 18 (LTS)
- Typescript
- Express.js (web framework)
- Prisma (ORM)

## Setup

1. Instalar requisitos:
   - npm (recomendado instalar com o nvm)

2. Instalar dependências:
   ```
   npm install
   ```

3. Criar o arquivo de váriaveis do ambiente:
   ```
   cp .env.example .env
   ```

4. Inicializar o Prisma:
   ```
   npx prisma generate
   ```
